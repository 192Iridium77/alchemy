# Portfolio

A Next.js portfolio website showcasing projects, companies, certifications, and recommended books.

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

1. Install dependencies:

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build

Build the project for production:

```bash
pnpm build
```

### Start Production Server

Start the production server:

```bash
pnpm start
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
